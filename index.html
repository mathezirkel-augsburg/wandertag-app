Disclaimer:

Diese App verwendet deinen Standort.

Beim Besuchen der "Tracking" Seite muss der Zugriff auf den Standort erlaubt werden, damit die App funktionieren kann.

Dieser wird aufgezeichnet und an einen zentralen Server geschickt.

Dabei gesammelten Daten werden nur im Mathezirkel ausgewertet und sind 100% anonym.

Außerhalb des Mathecamp läuft kein Server zur Datenaufzeichnung.

Die App hat nur Zugriff auf deinen Standort, während sie geöffnet ist, niemals im Hintergrund.

Installation:

Die App kann im Browser geöffnet werden. Dann ist eine Internetverbindung erforderlich (verbraucht sehr wenige Daten).

Wahlweise kann die App auch installiert werden und ist dann komplett offline verfügbar.

Es empfiehlt sich, vor dem Start den "Hard Update App" Button zu verwenden um die neusten Referenzdaten zu laden.

Position-Push:

Alle par Minuten versucht die App die gesammelten Daten zur späteren Analyse an den Server zu senden.

Dies sollte quasi keine mobilen Daten verbrauchen.

Solltest du dennoch Bedenken haben, kannst du mobile Daten während der App benutzung einfach ausschalten.
